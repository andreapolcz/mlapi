Applications

API that enables the creation and administration of third party applications.

Attributes

id Applications unique identifier.
site_id Country identifier of the application.
name Applications commercial name.
description Applications description.
thumbnail Applications thumbnail.
owner_id Applications owner identifier.
catalog_product_id Catalogue Product ID related to the application.
item_id Item ID related to the application.
need_authorization Indicates whether the application needs to be authorized by users. (Cant edit this).
short_name Applications access name. Part of the url.
url URL to access the application.
callback_url URL where the application is located.
sandbox_mode Sandbox mode. End users cant access the application.
active Indicates whether the application is active. (Cant edit this).
max_requests_per_hour Maximum number of allowed requests to the public API per hour. (Cant edit this).
secret_key Applications secret key.
scopes Scopes handled by the application.

Allowed methods

  • POST /applications

    Creates a new application.

  • PUT /applications/:id

    Modifies an application.

  • DELETE /applications/:id

    Deletes an application.

  • GET /applications

    Returns all applications.

  • GET /applications/:id

    Returns an application.

  • GET /applications/search

    Performs searches on applications.
