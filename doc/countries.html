País

List of countries

Attributes

id Country ID (ISO 3166-1 alpha-2)
iso_code Country ID (ISO 3166-1 alpha-2)
name Country name
status Active / Inactive status of the country in ML
locale Language ID (ISO code)
currency_id Currency ID (ISO code)
decimal_separator Decimal separator
thousands_separator Thousands separator
time_zone Time zone
geometry
Datos de geoposicionamiento
location Punto interior del polígono
states List of states in country

Allowed methods

  • GET /countries

    Returns all countries

  • GET /countries/:id

    Returns detailed information (list of provinces, etc.) of a country
